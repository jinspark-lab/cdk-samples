## How to Start EKS-App

1. cd kubernetes
2. cdk8s synth
3. cd ../
4. cdk synth
5. cdk deploy

If you are first to use CDK, you need to run CDK BootStrap.

How to install CDK?
 : [AWS Doc](https://docs.aws.amazon.com/cdk/v2/guide/getting_started.html)

How to install CDK8s?
 : [CNCF Doc](https://cdk8s.io/)
